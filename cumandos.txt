docker compose down
--rmi all → Remove todas as imagens criadas pelo Compose.

--volumes → Remove todos os volumes associados aos serviços (atenção: dados serão perdidos).

--remove-orphans → Remove containers órfãos que não estão definidos no arquivo docker-compose.yml.

docker compose down --rmi all --volumes --remove-orphans
